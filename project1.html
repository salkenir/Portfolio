Coding for the Web

I started working with websites my sophomore year of college, but it was mainly just designing through websites that do a lot of the behind the scenes work for you, like Squarespace or Wordpress.com. In 2021 I started hand-coding using visual studio code. I can code in HTML5 and I worked on 3 coding projects this year. I created an information travel guide website, a retail site, and this portfolio site.

Coding was an intimidating task, but I have come to really enjoy it and am surpised how fast it became natural to me. I love the freedom hand-coding gives you to be as in control and creative as possible. I would love to code webpages for clients in the future as a part of my career.

Starter Site

This is the first site I coded for a project. I had to create a "travel guide" kind of like a tourist site for Harrisonburg virginia. It had to feature a homepage about Harrisonburg, Downtown, JMU, and a photos page. Being my first site there is still room for improvements, but I like seeing my progress since I started. All the images used in this site had to be my own.


Retail Site

My most recent coding project was a retail site for a skincare brand I made up. For this project we had more flexibility with images we could use. I wanted to go for a natural skincare company and I wanted a soft, nuetral, but sleek design. I wanted the site to give off a 'clean' and 'trendy' feel.

Responsive Layout

For this project I learned how to do mobile-first coding and responsive layouts meaning the pages change to fit whichever device is being used. For my retail site I coded for mobile devices, tablets, and desktop screens.

I wanted a simple design when it came to what was 'clickable' on the retail site, so I went with a simple black outline when you hover your mouse over the 'add to cart' button.
